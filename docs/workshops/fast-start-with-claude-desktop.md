# Fast Start with Claude Desktop

It was great meeting you at the meetup!  This page is a quick guide to
getting you started building your own intelligent textbook with Claude.

## Where and When We Meet

We meet at **The Thinking Spot** on **Mondays from 4pm to 6pm** for short
getting-started sessions.

I am also happy to set up a **1-hour Google Meet** to get you started one
on one.

## What You Need Before We Start

To get started I strongly suggest you have the following two things ready:

1. A **Claude Pro account** set up ($20/month)
2. A **rough draft of a course description**

That is all you need.

## What We Will Walk Through Together

We will walk you through the following steps:

1. Create a GitHub account
2. Create a book repository on GitHub
3. Install Claude Code Desktop and Terminal
4. Install our book skills
5. Convert the draft course description into a high-quality description
6. Begin to generate the full textbook step by step

## What Happens After the First Meeting

After this initial meeting I check in weekly for a month, or on an
as-needed basis.

In general, after two months you will never need me again except for
emotional support.  Claude will be able to answer all your questions.
