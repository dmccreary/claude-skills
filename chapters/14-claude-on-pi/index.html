    Running Claude on the Raspberry Pi

    This guide covers setting up Claude Code on a Raspberry Pi 5 running Raspberry Pi OS.

    Step 1: Install Node.js and npm

    Claude Code requires Node.js. Install it using the NodeSource repository:

    1
2
    curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

    Verify the installation:

    1
2
    node --version
npm --version

    Step 2: Install Claude Code

    Install Claude Code globally using npm:

    1
    npm install -g @anthropic-ai/claude-code

    Step 3: Configure Copy/Paste Keyboard Shortcuts

    By default, the terminal uses Ctrl+Shift+C and Ctrl+Shift+V for copy/paste, which differs from standard desktop shortcuts. You can configure LXTerminal to use Super+C and Super+V instead, making it easier to use alongside other applications.

    Edit the LXTerminal configuration file:

    1
    nano ~/.config/lxterminal/lxterminal.conf

    Find the [shortcut] section and modify these lines:

    1
2
3
    [shortcut]
copy_accel=<Super>c
paste_accel=<Super>v

    Save the file and restart LXTerminal for changes to take effect.

    The full shortcut section should look like:

    Step 4: Install Miniconda (Optional)

    Miniconda provides a lightweight Python environment manager, useful for managing Python dependencies for various projects.

    Download and install Miniconda for ARM64:

    1
2
3
    wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-aarch64.sh -O /tmp/miniconda.sh
bash /tmp/miniconda.sh -b -p $HOME/miniconda3
rm /tmp/miniconda.sh

    Initialize conda for your shell:

    1
    ~/miniconda3/bin/conda init bash

    Restart your terminal or run:

    1
    source ~/.bashrc

    Verify the installation:

    1
    conda --version
